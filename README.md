# Yokai-medal_manager<br>
https://toy.bandai.co.jp/ja/series/youkai/youkai_ziten/
このURL専用のgoogle chromeの拡張機能です。
主な機能としては、妖怪メダルを持っているかどうかの管理、妖怪メダルを検索するためのフィルター、そしてフィルター機能のみを使用可能にできる拡張機能のオン、オフボタンです。
使いやすいインターフェースにしたため、初めての人でも簡単に使うことができると思います。
これはgoogle chromeのみで動作します。
拡張機能の入れ方としては、google chromeを開き、右上の三点ボタンから拡張機能という欄をクリックします。そしたら右上のデベロッパーモードを有効にし、有効化したら表示されるパッケージ化されていない拡張機能を読み込むというボタンを押し、フォルダを丸ごと選択します。
